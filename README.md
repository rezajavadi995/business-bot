# Telegram Business Bot (Ubuntu 24)

یک ربات بیزینسی کامل با پنل ادمین مدرن، قابلیت روشن/خاموش کردن فیچرها، مدیریت پیام آفلاین و نصب سریع.

## نصب سریع (One-liner)

```bash
bash -c "$(curl -fsSL https://raw.githubusercontent.com/rezajavadi995/business-bot/main/install.sh)"
```

> اگر ریپو private است، ابتدا پروژه را clone کنید و داخل فولدر پروژه اجرا کنید:

```bash
chmod +x install.sh && ./install.sh
```

## اجرای ربات

```bash
source .venv/bin/activate
python bot.py
```

## تنظیمات `.env`

```env
BOT_TOKEN=توکن_ربات_از_BotFather
ADMIN_ID=آیدی_عددی_ادمین
```

## منو و راه‌اندازی اولیه

1. `/start`
2. پیام `menu` برای باز شدن پنل.
3. سه گزینه اصلی:
   - ست ادمین
   - ست توکن
   - فعال‌سازی/روشن کردن ربات
4. مقادیر مهم در `.env` و `data.json` ذخیره می‌شوند.

## 10 فیچر ادمین

1. روشن/خاموش کردن کل ربات (`bot_active`)
2. ست ادمین
3. ست توکن (راهنمای تنظیم امن در `.env`)
4. تغییر پیام آفلاین
5. گزارش سریع (Uptime/IP/CPU/RAM/Disk/Users)
6. برودکست (`/broadcast`)
7. آمار کاربران
8. سوییچ تک‌تک فیچرها ON/OFF
9. لیست بازدیدکننده‌ها (در `data.json`)
10. سیستم لاگ (`logs/bot.log`)

## 10 فیچر بازدیدکننده

1. پیام خوش‌آمد
2. دکمه `menu`
3. پاسخ خودکار آفلاین (قابل تغییر توسط ادمین)
4. راهنما
5. خدمات
6. تعرفه
7. ساعات کاری
8. ثبت بازدیدکننده
9. احترام به وضعیت روشن/خاموش ربات
10. طراحی دکمه‌ای مدرن با inline keyboard

## کامندهایی که در BotFather ست کنید

- `/start` - شروع کار با ربات
- `/broadcast` - ارسال پیام همگانی توسط ادمین

## لاگ‌گیری و خطا

- لاگ‌ها در `logs/bot.log` ذخیره می‌شود.
- اسکریپت نصب `set -euo pipefail` دارد، یعنی خطاها silent نمی‌شوند.

## نکات توسعه

- دیتا در `data.json` ذخیره می‌شود.
- برای تغییر متن‌های خدمات/تعرفه/ساعات، فایل `data.json` را ویرایش کنید.
